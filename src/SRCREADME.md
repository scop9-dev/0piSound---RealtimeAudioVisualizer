# 0piSound Source Code

This folder contains the C# source code of 0piSound.

- All logic and UI are implemented in `Form1.cs`.  
- There is no `Program.cs`; the application starts directly from Form1.  
- Includes Visual Studio solution and project files (`0piSound.sln` and `0piSound.csproj`).  
- `app.manifest` is included for execution.

Dependencies: NAudio, Newtonsoft.Json, Costura.Fody.
